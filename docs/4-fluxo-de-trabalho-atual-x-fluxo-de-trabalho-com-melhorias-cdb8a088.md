# 4. Fluxo de trabalho atual X Fluxo de trabalho com melhorias

Insira aqui um fluxo de trabalho atual (o cliente deve fornecer o fluxo de trabalho já desenhado) e após a realização do discovery, remodele se preciso o fluxo de trabalho apresentado, incluindo melhorias. Também pode ser inserido a jornada do usuário, para que o time de desenvolvimento tenha claro o comportamento do produto no momento de desenvolvimento das telas.

Para modelagem dos fluxos deve ser utilizado o Lucid ou Miro, deve ser exportada a imagem em jpg e inserido o link do documento para possíveis edições e atualizações do fluxo.
