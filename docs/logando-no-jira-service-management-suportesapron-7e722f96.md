# Logando no Jira Service Management ( suportesapron

> Para que pudéssemos manter o serviço da Atlassian ainda no plano free, criamos uma conta para multiusuários, abaixo segue as instruções para acessar  a conta. **Atenção:** Para fazer comentários nos suportes, é preciso se identificar  na mensagem. Ex: " Fulano aqui, bom dia ….."

[Link para saber mais](https://youtu.be/7VTVkyu-Pf8) sobre a ferramenta.

> \

Portal do Cliente: [Link](https://suportesapron.atlassian.net/servicedesk/customer/portals)

Portal do Agente: [Link](https://suportesapron.atlassian.net/jira/projects)

Passo a passo para logar:

1- Entre no Vault, e acesse a pasta [/Sapron/Jira Service Management](https://vault.sapron.com.br/ui/vault/secrets/secret/show/Sapron/%20Jira%20Service%20Management)  para ter acesso às credenciais da ferramenta.

 ![Untitled](/api/attachments.redirect?id=52d1c1a8-0cda-43df-a9d4-8b8ad297acc5)

2- Após clicar no link da ferramentas, você será redirecionado para a tela de login. Aqui não é necessário entra com o Google, pois a conta de usuário foi feita  utilizando o e-mail do suportesapron@seazone.com.br diretamente. Para evitar a necessidade de se logar com o e-mail.

 ![Untitled](/api/attachments.redirect?id=a1a076b9-24f0-47ae-b673-81f7a4d59547)

3- Insira o e-mail que está no Vault. Clique em 'Continue'.

 ![Untitled](/api/attachments.redirect?id=08e834fc-b7a3-42df-9ea4-2ff1a0bccb56)

4- Insira a senha que está no Vault. Clique em 'Entrar'.

 ![Untitled](/api/attachments.redirect?id=eec67808-14f4-4574-9f9f-03724110edd2)

5- Finalizou o login. Agora escolha o projeto Suporte Sapron Management
