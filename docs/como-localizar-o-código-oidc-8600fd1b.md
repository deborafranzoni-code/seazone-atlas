# Como localizar o código OIDC

## Para Que Serve o Código OIDC?

O código OIDC é seu **identificador único** no Vault. Você precisa dele para:

* **Solicitar acesso** a novas pastas/secrets
* **Preencher formulários** de suporte
* **Identificar sua conta** em requisições 

**🔑 Sem esse código, a equipe não consegue liberar permissões para você.**


---

## Como Encontrar Seu Código OIDC

### Passo 1: Acessar o Menu de Usuário


1. Faça login no Vault
2. No canto **superior direito** da tela, localize o ícone de **pessoa** (👤)
3. Clique no ícone de pessoa

 ![](/api/attachments.redirect?id=d88a169e-d1b0-4a91-a068-1f6da1c14cfd " =952x495")

### Passo 2: Visualizar Seu Código

Um menu suspenso abrirá mostrando:

 ![](/api/attachments.redirect?id=0c949af0-2d17-40b6-ab0a-f8e7e89275c1 " =1063x593")

**A primeira linha é o seu código OIDC completo.**

No exemplo acima: `Oidc-111515204765644000323`

### Passo 3: Copiar o Código

**Método 1 - Seleção manual:**


1. Clique e arraste o mouse sobre o código `Oidc-XXXXX...`
2. Pressione `Ctrl+C` (Windows/Linux) ou `Cmd+C` (Mac)


---

## FAQ

**Meu código OIDC muda?** → Não. Ele é permanente enquanto você usar a mesma conta SSO corporativa.

**Outras pessoas podem ter o mesmo código?** → Não. Cada código OIDC é único por usuário.

**Esqueci de anotar meu código. Onde encontro de novo?** → Basta fazer login no Vault e clicar no ícone de pessoa (👤) novamente. O código sempre estará lá.

**Preciso decorar meu código?** → Não é necessário. Você sempre pode consultar na interface quando precisar.

**O código muda se eu trocar de time/departamento?** → Não. O código permanece o mesmo. Apenas suas permissões de acesso podem mudar.
