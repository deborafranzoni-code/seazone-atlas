# check_in_time

Descrição: Hora exata do dia para check-in na propriedade. O formato HHMM, HH:MM ou HH:MM:SS é aceito. Por exemplo, use 14:00 em vez de 14:00 ou 14:00. Nenhum outro formato será analisado nem exibido.
