# Alerta de Picos de Demanda

É um sistema de inteligência de mercado projetado para ser nosso **radar de oportunidades**. Ele identifica e alerta a equipe sempre que detecta um aumento súbito e anormal na procura por aluguéis para uma região e um período futuros específicos.
