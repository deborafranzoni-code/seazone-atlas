### Como faço para que um banner seja exibido?

* Acesse a aba **["Home - Banners"](https://seazone-reservas-production-cms.prismic.io/builder/pages/ZMzyexEAACIAbahV?fallback=YiUzRHdvcmtpbmclMjZjJTNEcHVibGlzaGVkJTI2bCUzRHB0LWJy&s=published&section=Home+-+Banners)**
* No canto esquerdo verifique as abas de variantes e veja se esta com a variante que você deseja
* Após adicionar a variante basta fazer o upload das imagens nos respectivos campos

### Como adicionar uma nova variante ?

Você pode clicar no + para adicionar uma nova variante para a home

 ![](/api/attachments.redirect?id=6072fe10-3e03-4e05-a9ff-505403294db8 "left-50")


**Alguns pontos importantes sobre as variantes:**

* A ordem na qual as variantes são importadas não tem impacto no website
* Não adicione duas vezes a mesma variantes pois apenas uma sera exibida no site

Após clicar no botão de adicionar, um modal irá aparecer na `tela` principal, nele basta selecionar a variante desejada

 ![](/api/attachments.redirect?id=afa6f77e-cdbd-4e01-8874-185cacf3ee44 " =844x636")

Basta selecionar a variante desejada e adicionar as imagens necessárias 

### Como mudar o banner que está sendo exibido?

* Acesse a aba **["Home - Banners"](https://seazone-reservas-production-cms.prismic.io/builder/pages/ZMzyexEAACIAbahV?fallback=YiUzRHdvcmtpbmclMjZjJTNEcHVibGlzaGVkJTI2bCUzRHB0LWJy&s=published&section=Home+-+Banners)**
* Nas imagens da variante selecionada clique no botão relplace no canto superior direito e selecione uma imagem nova

### **Formato de imagens aceitos:** **Proporções das imagens que devem ser usadas:**

* Desktop: `1200x300`
* Mobile: `360x150` </aside>

### Como adicionar os banners para a nova home ou versões futuras do website ?

Para a nova home e próximas versões serão adicionadas sempre novas variantes no prismic, elas podem ser acessadas pela seção à esquerda descrita no tópico **==Como adicionar uma nova variante==**


Após abrir o modal de seleção de variantes selecione a variante de nova home, ou a Nova home Promo para adicionar banners à area de promoção da nova home do website

 ![](/api/attachments.redirect?id=bfc1fbd8-9635-4170-b827-17b27ad3a0ed)

Agora basta adicionar uma imagem para o desktop e outra para o mobile assim como é feito na home atual

 ![](/api/attachments.redirect?id=5d5bc8c6-d3b6-4822-a6ad-3836fe4867ec)

Diferente da home antiga, a nova home permite que seja adicionado um link diferente para cada banner, basta adicionar no campo `href`, caso não tenha necessiade de ser um link diferente basta colocar o mesmo link em todos os href. 

**obs: caso não seja informado o href o banner não redirecionara o usuário para nenhum lugar ao clicar**


Para a home nova é possível também adicionar vários banners diferentes, para adicionar mais banners basta clicar no botão `+ Add item in "Desktop"`

 ![](/api/attachments.redirect?id=7c963e30-f013-4aec-aeb0-54ab655d0ad9)

Após clicar basta adicionar a nova imagem e o novo link, caso precise,  também é possível ordenar os banners, a ordem na qual estiverem no prismic sera a ordem na qual vão aparecer no site, sendo o primeiro o banner principal.
