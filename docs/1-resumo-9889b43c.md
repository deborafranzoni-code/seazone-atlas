# 1. Resumo

## **1.1 Introdução e objetivo**

O processo de pré check-in foi pensado para facilitar a entrada do hóspede nos imóveis locados sem que o anfitrião tenha que solicitar em seu primeiro contato as informações pessoais e dados relevantes a reserva. Para que este processo se torne funcional, é necessário a refatoração da tela de pré check-in, card de controle do anfitrião e completar reservas do atendimento.

## **1.2 Para quem é?**

As oportunidades apresentadas são pensadas para os seguintes times:

Atendimento: maior visibilidade do processo do pré check-in, sendo possível visualizar se o link foi enviado e preenchido Anfitrião: consiga visualizar os dados dos hóspedes facilitando o seu fluxo de trabalho Hóspede: maior rapidez no processo de check-in

## **1.2 Glossário**

* **Franqueado/Anfitrião:** pessoa responsável por realizar o check-in do cliente e verificar os dados
* **Atendimento:** responsável por enviar o link de pré check-in ao hóspede
* **Hóspede:** responsável pelo correto preenchimento e envio de todos os dados solicitados
