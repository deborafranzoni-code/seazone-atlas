# Como adicionar Dados de Contato

[Como adicionar ou excluir Dados de Contato.pdf](Como%20adicionar%20Dados%20de%20Contato%201b47b1fbe3d34fe098b0f8d6212201fc/Como_adicionar_ou_excluir_Dados_de_Contato.pdf)

Este guia fornece instruções passo a passo sobre como adicionar ou excluir dados de contato no Sapron. Siga as etapas simples e rápidas para adicionar seus dados de contato e garantir que estejam atualizados e acessíveis para nossa equipe de atendimento.


1. Acesse o site <https://sapron.com.br/login>
2. Insira seu login e senha e clique em "Entrar".

3. Clique no ícone de acesso ao perfil no canto superior direito.

 ![https://ajeuwbhvhr.cloudimg.io/colony-recorder.s3.amazonaws.com/files/2023-12-11/22c57ddf-66e5-496b-a5b2-eb1a1663fad6/user_cropped_screenshot.jpeg?tl_px=0,0&br_px=1001,721&force_format=png&width=1120.0&wat=1&wat_opacity=0.7&wat_gravity=northwest&wat_url=https://colony-recorder.s3.us-west-1.amazonaws.com/images/watermarks/FB923C_standard.png&wat_pad=970,54](https://ajeuwbhvhr.cloudimg.io/colony-recorder.s3.amazonaws.com/files/2023-12-11/22c57ddf-66e5-496b-a5b2-eb1a1663fad6/user_cropped_screenshot.jpeg?tl_px=0,0&br_px=1001,721&force_format=png&width=1120.0&wat=1&wat_opacity=0.7&wat_gravity=northwest&wat_url=https://colony-recorder.s3.us-west-1.amazonaws.com/images/watermarks/FB923C_standard.png&wat_pad=970,54)


4. Clique em "Meus dados".

 ![https://ajeuwbhvhr.cloudimg.io/colony-recorder.s3.amazonaws.com/files/2023-12-11/7b1b1f9b-b2db-4015-9a45-f6ffdff23fdc/user_cropped_screenshot.jpeg?tl_px=0,0&br_px=1005,721&force_format=png&width=1120.0&wat=1&wat_opacity=0.7&wat_gravity=northwest&wat_url=https://colony-recorder.s3.us-west-1.amazonaws.com/images/watermarks/FB923C_standard.png&wat_pad=970,215](https://ajeuwbhvhr.cloudimg.io/colony-recorder.s3.amazonaws.com/files/2023-12-11/7b1b1f9b-b2db-4015-9a45-f6ffdff23fdc/user_cropped_screenshot.jpeg?tl_px=0,0&br_px=1005,721&force_format=png&width=1120.0&wat=1&wat_opacity=0.7&wat_gravity=northwest&wat_url=https://colony-recorder.s3.us-west-1.amazonaws.com/images/watermarks/FB923C_standard.png&wat_pad=970,215)


5. Vá até a seção "Telefones para contato" e clique no ícone de lixeira ao lado dos contatos que deseja excluir.

 ![https://ajeuwbhvhr.cloudimg.io/colony-recorder.s3.amazonaws.com/files/2023-12-11/5d163c24-46e5-4f5e-aa32-c17423c8d79e/user_cropped_screenshot.jpeg?tl_px=0,0&br_px=1920,599&force_format=png&width=1120.0&wat=1&wat_opacity=0.7&wat_gravity=northwest&wat_url=https://colony-recorder.s3.us-west-1.amazonaws.com/images/watermarks/FB923C_standard.png&wat_pad=858,95](https://ajeuwbhvhr.cloudimg.io/colony-recorder.s3.amazonaws.com/files/2023-12-11/5d163c24-46e5-4f5e-aa32-c17423c8d79e/user_cropped_screenshot.jpeg?tl_px=0,0&br_px=1920,599&force_format=png&width=1120.0&wat=1&wat_opacity=0.7&wat_gravity=northwest&wat_url=https://colony-recorder.s3.us-west-1.amazonaws.com/images/watermarks/FB923C_standard.png&wat_pad=858,95)


6. Clique em "Excluir".

 ![https://ajeuwbhvhr.cloudimg.io/colony-recorder.s3.amazonaws.com/files/2023-12-11/80420083-94dc-4dd8-ba25-b4e7fa40aca5/ascreenshot.jpeg?tl_px=516,171&br_px=1806,893&force_format=png&width=1120.0&wat=1&wat_opacity=0.7&wat_gravity=northwest&wat_url=https://colony-recorder.s3.us-west-1.amazonaws.com/images/watermarks/FB923C_standard.png&wat_pad=524,284](https://ajeuwbhvhr.cloudimg.io/colony-recorder.s3.amazonaws.com/files/2023-12-11/80420083-94dc-4dd8-ba25-b4e7fa40aca5/ascreenshot.jpeg?tl_px=516,171&br_px=1806,893&force_format=png&width=1120.0&wat=1&wat_opacity=0.7&wat_gravity=northwest&wat_url=https://colony-recorder.s3.us-west-1.amazonaws.com/images/watermarks/FB923C_standard.png&wat_pad=524,284)


7. Para adicionar novo contato, vá até o final da seção e clique em "+Adicionar novo contato".

 ![https://ajeuwbhvhr.cloudimg.io/colony-recorder.s3.amazonaws.com/files/2023-12-11/04287538-baf3-414b-9f06-ab113bfe8ab3/user_cropped_screenshot.jpeg?tl_px=0,0&br_px=1920,893&force_format=png&width=1120.0&wat=1&wat_opacity=0.7&wat_gravity=northwest&wat_url=https://colony-recorder.s3.us-west-1.amazonaws.com/images/watermarks/FB923C_standard.png&wat_pad=532,441](https://ajeuwbhvhr.cloudimg.io/colony-recorder.s3.amazonaws.com/files/2023-12-11/04287538-baf3-414b-9f06-ab113bfe8ab3/user_cropped_screenshot.jpeg?tl_px=0,0&br_px=1920,893&force_format=png&width=1120.0&wat=1&wat_opacity=0.7&wat_gravity=northwest&wat_url=https://colony-recorder.s3.us-west-1.amazonaws.com/images/watermarks/FB923C_standard.png&wat_pad=532,441)


8. Digite os dados do contato que deseja adicionar.

 ![https://ajeuwbhvhr.cloudimg.io/colony-recorder.s3.amazonaws.com/files/2023-12-11/55ee3f1a-4443-4f7c-b884-77e7e03996da/user_cropped_screenshot.jpeg?tl_px=0,158&br_px=1290,879&force_format=png&width=1120.0&wat=1&wat_opacity=0.7&wat_gravity=northwest&wat_url=https://colony-recorder.s3.us-west-1.amazonaws.com/images/watermarks/FB923C_standard.png&wat_pad=246,277](https://ajeuwbhvhr.cloudimg.io/colony-recorder.s3.amazonaws.com/files/2023-12-11/55ee3f1a-4443-4f7c-b884-77e7e03996da/user_cropped_screenshot.jpeg?tl_px=0,158&br_px=1290,879&force_format=png&width=1120.0&wat=1&wat_opacity=0.7&wat_gravity=northwest&wat_url=https://colony-recorder.s3.us-west-1.amazonaws.com/images/watermarks/FB923C_standard.png&wat_pad=246,277)


9. Ao final, clique em "Salvar".
