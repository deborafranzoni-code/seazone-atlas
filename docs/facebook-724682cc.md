### Token de Acesso

Um token de acesso fornece a você o acesso à API de Conversões. Sempre que você fizer uma chamada de API, será necessário usar seu token de acesso.

Gere seu token de acesso e copie-o abaixo. Para sua segurança, o Facebook não salvará seu token de acesso. Se esquecer seu token de acesso, você poderá criar outro.

[Saiba mais](https://developers.facebook.com/docs/marketing-api/conversions-api#access-token)
