# Discovery - Automatização de Gestão de Contas

O objetivo deste documento é mapear todo o processo de gestão de contas e entender possíveis pontos para automatização. 

Os principais fluxos a serem realizados por gestão de contas são: 

* Registro de novas propriedades terão gestão de contas
* Coleta mensal de conta de forma manual
* Centralizar informações das contas na planilha 
* Enviar para pagamento
