# check_out_time

Descrição: Hora exata do dia para o check-out na propriedade. O formato HHMM, HH:MM ou HH:MM:SS é aceito. Por exemplo, use 11:00 em vez de 11:00 ou 11:00. Nenhum outro formato será analisado nem exibido.
