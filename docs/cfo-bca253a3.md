# **==__________________Sobre_______________________==**

## *==———Descrição———————————-==*

* Objetivo dessa planilha é consolidar os valores da área financeira

## *==———Modificação——————————-==*

* Troca das tabelas do Sapron de "Property Pay" para "Closing", para os código `metabaseChurnReceitasFuturas` e `metabaseReceitaLimpezaDireta`

## ==———A**ba GMV**==

### `metabaseChurnReceitasFuturas`

* **Objetivo**: puxar o extrato dos imóveis
* **Base de Dados**
  * **Metabase (v3)**
    * Closing Property Resume
    * Closing Host Resume
    * Property Property

### `metabaseReceitaLimpezaDireta`

* **Objetivo**: puxar os valores de limpeza e receita
* **Base de Dados**
  * **Metabase (v3)**
    * Closing Property Resume
    * Reservation Reservation
    * Reservation Ota
    * Property Property
    * Closing Host Resume
