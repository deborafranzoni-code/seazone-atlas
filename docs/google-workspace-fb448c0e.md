# Google Workspace

<aside> 💡 O google workspace .é o aplicativo de administração do nosso workspace no google, ele é utilizando frequentemente para recuperação e criação de usuários internos e também para configurações específicas

</aside>

### Criar contas google

1 - Na tela principal do google admin clique em **Adicionar um usuário**

 ![Captura de tela 2024-06-18 111230.png](/api/attachments.redirect?id=5c9da894-6d4c-4960-8a13-bcd85070bff5)

2 - Adicione as informações do usuário a ser criado

> lembre-se que o padrão atual de criação de emails na seazone consiste em \[letra inicial do nome + . + sobrenome\]

 ![Captura de tela 2024-06-18 111410.png](/api/attachments.redirect?id=621c396c-7ce8-4619-a937-235f8a50039a)

3 - Clique em **redefinir senha** para gerar uma senha temporária para aquela conta

> Essa senha serve somente para que o usuário acesse a primeira vez , após isso ele será obrigado a definir  a senha oficial da conta

 ![Captura de tela 2024-06-18 111707.png](/api/attachments.redirect?id=a878a74d-cde3-4977-a0f2-b840dd84e3b8)

4 - Entre na aba de **segurança** do perfil  clique na opção **verificação de duas etapas** e gere um **código alternativo**

> Esse código assim como a senha deve ser utilizado somente para o primeiro acesso, após isso o usuário deve cadas um dispositivo de 2FA na conta e entrar com os códigos de autenticação

 ![Captura de tela 2024-06-18 112242.png](/api/attachments.redirect?id=3fa8546d-667d-4043-91c5-95e85e4a3975)

5 - Envie os dados para o solicitante ✅

<aside> 🚨

Para criar contas de franqueados o domínio correto a ser utilizado é o **seazonefranquias.com.br**

</aside>

## 3 - Envie a senha para o solicitante

> O ideal é que o solicitante tenha um dispositivo de 2FA configurado e consiga fazer o acesso sem a necessidade de um código alternativo , porém a realidade não é ideal então se for necessário acesse a aba de segurança clique na opção **Verificação de duas etapas** e gere o **código alternativo** para o solicitante

### Remover contas

1 - Na aba principal selecione a opção **excluir usuário**

 ![Captura de tela 2024-09-09 160414.png](/api/attachments.redirect?id=fcbb090f-e8a6-4dec-af0f-4f01a15b5d4f)

2- Digite o nome ou email do usuário que deseja remover

 ![Captura de tela 2024-09-09 160619.png](/api/attachments.redirect?id=82baf2d2-0f2b-4848-9236-88c171be2b42)

3 - Na tela de exclusão escolha de deseja transferir os dados do usuário que será removido para outro

 ![Captura de tela 2024-09-09 160755.png](/api/attachments.redirect?id=5f6e1405-8caf-47fa-9259-92c33230e246)

4 - Role a página até o final e clique em **excluir usuário**

<aside> 🚨

Geralmente o líder do colaborador em questão pode ser abordado para saber se os dados do mesmo precisam ser transferidos pra alguém ou podem ser apagados

</aside>

## Custos 💸

Atualmente, temos a assinatura do plano Enterprise Starter, cujo custo por usuário varia ao longo do tempo. A tabela abaixo mostra o valor por usuário até 2026, quando o preço se tornará fixo.

| **Data** | **Custo por usuário** |
|----|----|
| 25/05/2024 a 25/05/2025 | R$ 44,95 |
| 25/05/2025 a 25/05/2026 | R$ 47,85 |
| Após  25/05/2026 | R$ 58,00 |

## Grade de responsabilidades 🫂

Atualmente, a administração do Google Workspace está dividida principalmente entre a governança e o time de People. Por isso, decidimos criar uma grade de responsabilidades que esclarece quem é responsável por cada atividade.

| Responsabilidade | Responsável |
|----|----|
| Criar contas de colaboradores | People |
| Criar contas de franqueados | Governança |
| Recuperar contas (franqueados/ colaboradores) | Governança |
| Remover contas  colaboradores | People |
| Remover contas franqueados | Governança |
| Gerenciar domínios | Governança |
| Permissão a usuários | Governança |
| Gerenciar grupos de usuários | Governança |
