# Mapa de Calor de Polígonos

O Mapa de Calor de Polígonos é uma aplicação analítica interativa desenvolvida para apoiar o time de Revenue Management (RM) na identificação de discrepâncias internas de mercado dentro de regiões (polígonos).

A ferramenta permite analisar, de forma visual e objetiva, se os grupos de imóveis, definidos pela combinação de Strata e Número de Quartos, estão coerentes entre si ou se existem outliers relevantes que indiquem oportunidades ou problemas de precificação, demanda ou faturamento.
