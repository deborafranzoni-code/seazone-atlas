# Configiuração

Envie convites de acesso para as pessoas que irão utilizar a Conta Simples

# Usuários

Na Conta Simples, você pode atribuir diferentes tipos de permissionamentos e acessos para seus colaboradores

# Defina um novo perfil de acesso

Ao criar um novo usuário é possível escolher qual o nível de acesso ele terá na plataforma. Só acesso a cartões específicos, poder criar novos cartões, acesso a todas as ferramentas. Controle e segurança para a empresa.

# Envie o convite

Com o perfil escolhido, informe os dados da pessoa que receberá o convite por e-mail.

Após enviar os convites, você já pode distribuir os cartões entre os colaboradores.
