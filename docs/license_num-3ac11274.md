# license_num

Descrição: Número de licença (turista ou comercial) necessário para exibição de propriedades em determinadas regiões do mundo. Pode ser repetido e, se houver várias licenças, sugerimos adicionar autoridade como contexto. Por exemplo: "Paris: 123456ABC".
