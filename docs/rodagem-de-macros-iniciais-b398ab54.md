# Rodagem de Macros Iniciais✅


\
**Descrição**


As Macros que carregam os códigos deverão ser rodadas para que os dados sejam puxados e compilados para a Conciliação e para o Fechamento.


\
**Prazo**


As Macros de Fechamento deverão ser rodadas **nos 3 primeiros dias de Fechamento** (após a rodagem das Macros Mensais de Conciliação)


\
**Macros**


* Acessar a aba Dashboard da Planilha Fechamento Mensal Template

[Fechamento Mensal Template - Aba Dashboard](https://docs.google.com/spreadsheets/d/1UoGdsSrhNuyeTupWzP3tgg1mKUF81X8Ac6rwspoTRD0/edit)


* Alterar a data da Planilha para a de referência do Fechamento


* Alterar o ID da planilha para o da Conciliação de referência (código que consta entre barras no link)


* Clicar no botão Imóvel

[🏠Conferências Fechamento Imóvel](/doc/conferencias-fechamento-imovel-JW7cd02eaB)


* Clicar no botão Proprietário

[💁🏻‍♂️Conferências Fechamento Proprietário](/doc/conferencias-fechamento-proprietario-RqRotINSpE)


* Clicar no botão Anfitrião

[👨🏻‍💻Conferências Fechamento Anfitriões](/doc/conferencias-fechamento-anfitrioes-QN0K1cqkxu)
