# Bancos de Dados

Alguns processos do time **–** tanto de **back-end** quanto de **front-end** **–** dependem de acesso direto ao banco de dados. É o caso, por exemplo, da **gestão de popups** e dos **termos de uso**, que exigem a leitura e edição de tabelas específicas.

Recentemente, em setembro/25 a forma de acesso aos bancos foi reformulada, então esta seção serve justamente para te guiar nesse novo processo: desde como preparar o ambiente local até como fazer um dump atualizado.

Abaixo, na seção de **DOCUMENTOS**, estão separadas as documentações específicas de cada processo. A sugestão é primeiro fazer o setup para conectar aos bancos de STG/PRD para depois realizar o dump ;)
