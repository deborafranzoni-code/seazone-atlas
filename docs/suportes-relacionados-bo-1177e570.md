# Suportes Relacionados  BO

### BO RESERVAS

Dizer que uma reserva teve BO, significa dizer que houve algum tipo de problema com aquela reserva. Os problemas podem ser gerados por uma infinidade de causas como:

* Problemas no imóvel → falta de água, luz, internet, mal funcionamento da geladeira, pia, fogão, ou banheira;
* Problemas com a limpeza → hóspede reclamou que o imóvel estava sujo;
* Problemas no check-in → anfitrião atrasou para fazer o check-in, ou reserva não foi enviada a tempo para o hotel;
* Problemas com barulhos → hóspede reclama de barulho de vizinhos ou obras perto do local;
* Divergência de anúncio → no anúncio dizia que tinha piscina, mas de fato não tem, localização errada, etc..

Esses problemas ocorrem todos os meses e acabam impactando na conciliação, visto que normalmente é emitido um reembolso ao hóspede como uma forma de "restituí-lo" pelo transtorno. Com isso, é preciso definir quem arcará com esses custos.
