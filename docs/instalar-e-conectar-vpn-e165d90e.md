# \*📖 Guia de instalação do cliente VPN pritunl

### Com o link (solicitar no fluxo do canal #suporte-governança) em mãos previamente passado por e-mail ou slack siga os passos a seguir:


1. Copie o link e cole em um navegador de sua preferência:
2. Feito isso receberá uma página parecida como a imagem abaixo:

   ![Profile Pritunl](/api/attachments.redirect?id=6a4ba1b8-9c0c-4ac7-8116-687d456ef602)

💡 OBS: o Link é válido apenas por 24hs após isso contate a equipe de Governança e abra um ticket para gerar um novo link de perfil para o seu usuário caso necessários.


## Pré requisitos:


1. **Google Authenticator** para ler o código QRcode
2. Pritunl cliente
3. Alterar pin na tela do perfil, se estiver alterando pela primeira vez o pin padrão é 123456.

## Instalar o Pritunl client

O download do Pritunl client deve ser feito via este [link](https://client.pritunl.com/#install). Na página de download você pode escolher o seu sistema operacional, baixar e instalar o cliente

 ![Download Pritunl](/api/attachments.redirect?id=464bc1c9-81ca-47c2-86a9-90488a5bdc69)

Após instalado o Client irá aparecer desta forma:

 ![Client Pritunl](/api/attachments.redirect?id=93ff078c-505f-4497-b68d-dac29e2d02f4)

## Conectar Pritunl

Para conectar, clique no campo import e cole a url do seu Profile e escolha a opção Import,  conforme imagem abaixo:

 ![Import Pritunl](/api/attachments.redirect?id=68199047-3925-41bf-aa87-89481627f448)

Import Pritunl

Feito isso será apresentado a tela a seguir para confirmar a importação bem sucedida.

 ![TelaClient Pritunl](/api/attachments.redirect?id=334143a0-60ec-4022-b4b3-f404ce67a3f4)

Tela Client Pritunl

 💡 Caso haja algum problema ao importar o perfil com a url, você pode fazer o download do perfil na url do seu perfil, escolher a opção import profile no pritunl client e inserir o arquivo baixado.


Agora ao escolher a opção Connect será apresentado a seguinte tela:

 ![Connect Pritunl](/api/attachments.redirect?id=309881cd-732d-475f-9fd5-51415b0dba57)

Connect Pritunl

Digite o pin que foi criado e o código de autenticação de 2 fatores no caso utilizando o Google Authenticator.

 ![Untitled](/api/attachments.redirect?id=5a83767d-eaed-43bf-a160-c090ad3b31c1)

Se tudo deu certo você receberá um ip interno do túnel para acesso ao ambiente VPN conforme a imagem e os campos destacados abaixo cliente address e o tempo de conexão.

Agora estamos conectados, pode testar seu acesso e tenha um bom trabalho!!
