# Realocação de Reserva não atualizou no Sapron

**Observações**

Precisa ter acesso ao Banco de Dados de Produção do Sapron.

**Contexto**

Nesse contexto o hospede tinha realizado uma reserva para um imóvel X e dias depois fez a realocação para o imóvel Y. Com isso o Sapron acabou não  atualizando a reserva para o novo imóvel.
