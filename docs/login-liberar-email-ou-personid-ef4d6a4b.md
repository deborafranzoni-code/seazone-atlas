# Login - Liberar email ou personID

Quando precisar liberar um email ou personID

Localizar o registro na tabela Tabela Account User

Alterar os campos na Tabela Account:


1. Email: de `emaildapessoa@gmail.com`  para >>  `szn_suport_emaildapessoa@gmail.com`
2. PersonID: de `123456789` para >> `suport123456789`
