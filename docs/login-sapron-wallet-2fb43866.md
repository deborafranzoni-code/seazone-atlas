# Login (Sapron - Wallet)

# Troca de email ou senha

Quando precise trocar e-mail ou senha dos usuarios do sapron ou wallet sga esses passos


1. Acesse o link segundo o ambiente

* Ambiente de ==Produção== PROD: <https://api.sapron.com.br/admin/account/user/>

  \
* Ambiente de Teste ==STG==: <https://api.staging.sapron.com.br/admin/account/user/>

*Se não tiver acesso, pode pedir com Gui ou seu lider*$$



2. Digite o e-mail e clic em "Buscar" 

 ![](/api/attachments.redirect?id=e679ab55-aec4-4e13-ae52-f1602c94bfb4 " =480x108")



3. Na seção password, clic no link "*this form"* e inserir uma nova senha, de preferencia que tenha, letras minusculas, maiúsculas, números e letras especiais.

 ![](/api/attachments.redirect?id=90975392-fbe7-412c-b20f-deb28fefc5f4 " =861x106")


\

3. Envie os dados de acesso para o relator do suporte, seguindo esse formato

**====DADOS DE ACESSO=========**\nNova senha e login para esse suporte `<adicione o link do slack onde foi solicitado o suporte>`\ne-mail  `<email_de_acesso>`\nsenha  **<nova_senha>**


\
