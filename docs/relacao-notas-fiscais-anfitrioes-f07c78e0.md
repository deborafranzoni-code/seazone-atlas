# Relação Notas Fiscais Anfitriões ✅


\
**Descrição**


A Seazone envia aos Anfitriões, com o intuito de auxiliá-los na emissão de suas notas, uma relação onde constam os dados dos Proprietários e valores correspondentes à comissão recebida (8% da receita dos imóveis).


\
**Prazo**


Até o dia 15 de cada mês


\
**Fluxograma do Processo**


Geração do Documento de Notas Fiscais

\
**Processo**


* Após a gerar e conferir os dados da aba Notas Fiscais na planilha Fechamento Mensal Template de acordo com as instruções na Página "Notas Fiscais Comissão Seazone"

[Fechamento Mensal Template](https://docs.google.com/spreadsheets/d/1UoGdsSrhNuyeTupWzP3tgg1mKUF81X8Ac6rwspoTRD0/edit)

[📜Notas Fiscais Comissão Seazone ✅](/doc/notas-fiscais-comissao-seazone-wlT0slu27b)


* Clicar no botão "Fechamento" situado no menu superior


* Clicar no botão "Anfitrião"


* Clicar no botão "Gerar DOCs de NF"


* Acessar Pasta Prestação de Contas no Drive 00 - PMS

[00 - PMS - Prestação de Contas](https://drive.google.com/drive/folders/1Pmm2T6sdM8vEngwWWnwtyQdvfC-0IzhJ)


* Selecionar a pasta com o ano e mês de Referência para o Fechamento


* Selecionar a pasta Anfitriões


* Os documentos serão gerados nesta pasta (permitindo edições manuais)


* Acessar os Documentos para conferir se todos os proprietários que possuem valor de NFs estão com os dados preenchidos


* Acessar novamente a Planilha Fechamento Mensal Template

[Fechamento Mensal Template](https://docs.google.com/spreadsheets/d/1UoGdsSrhNuyeTupWzP3tgg1mKUF81X8Ac6rwspoTRD0/edit)


* Clicar novamente no botão "Fechamento" situado no menu superior


* Clicar novamente no botão "Anfitrião"


* Clicar no botão "Transformas todos DOCs em PDFs"


* Acessar novamente a Pasta Prestação de Contas no Drive 00 - PMS

[00 - PMS - Prestação de Contas](https://drive.google.com/drive/folders/1Pmm2T6sdM8vEngwWWnwtyQdvfC-0IzhJ)


* Selecionar a pasta com o ano e mês de Referência para o Fechamento


* Selecionar a pasta Anfitriões


* Selecionar a pasta PDFolder


* Selecionar os documentos um a um


* Acessar o e-mail do Administrativo Seazone


* Escrever um novo e-mail utilizando o modelo pré-estabelecido para relação de Notas Fiscais


* Alterar o nome e a data no título e corpo do e-mail


* Selecionar o e-mail do anfitrião desejado e o e-mail anfitriões@seazone


* Anexar o relação de NFs para envio


\
**Lembrete de Exceção**


Os casos de Nota Fiscal para Pessoa Física não tem a emissão obrigatória, contudo, na relação de do anfitrião Erion Xhafaj é preciso adicionar a observação de "Nota Fiscal Obrigatória" para o propritetário Paulo Cesar Echeverria (TPS109) que deseja recebê-la sempre .
