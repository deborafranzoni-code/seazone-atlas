# Levantamento de KPIs

Esta seção tem como objetivo descrever como funcionam o levantamento de KPIs que utilizamos hoje no Sapron.

### 1° Tasks em Backlog

Para a coleta de dados desse KPI, é necessário seguir os seguintes passos:


1. Acessar o Jira Software através desse link: <https://seazone.atlassian.net/jira/dashboards/10002>
2. Já no primeiro quadro chamado: **Issue Statistics: Sapron Delivery (Status)**
3. Verificar quantas tasks estão em: "to do".

### 2° Tasks em doing/testing/staging


1. Acessar o Jira Software através desse link: <https://seazone.atlassian.net/jira/dashboards/10002>
2. Já no primeiro quadro chamado: **Issue Statistics: Sapron Delivery (Status)**
3. Verificar quantas tasks estão em: "In progress", "PR Approved", "Review PR" e "Staging".

### 3° PRs entregues na semana Backend/Frontend


1. Acessar o Jira Software através desse link: <https://seazone.atlassian.net/jira/software/projects/SAP/boards/3>
2. Clicar em: "Filters" e selecionar: "Filtros para KPI tasks last week".

### 5° Deploys na semana


1. Acessar o Git Hub.
2. Clicar em Sapron-PMS-Web
3. Clicar em Pull Requests
4. Clicar em "closed"
