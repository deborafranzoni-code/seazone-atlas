# Alerta de Preços Anomalos

É um sistema de monitoramento automático projetado para ser o **guardião da nossa estratégia de precificação**. Ele identifica e notifica o time sempre que o preço de um imóvel se desvia significativamente do que é considerado um valor realista e competitivo para uma determinada data.
