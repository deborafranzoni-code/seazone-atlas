# DRE Gerencial

O [DRE Gerencial](https://docs.google.com/spreadsheets/d/1qmUwZHJTIjBhPjK_ySlRDloQMIFbPyhCFRSnMcpTHQs) tem como objetivo relacionar nossas despesas x receitas de forma categorizada.

A atualização do DRE Gerencial é feita para as empresas Seazone Serviços, Seazone Investimentos e Khanto Reservas na mesma aba de movimentações.

Portanto é necessário copiar os dados das saídas das três sheets AdmSys.
