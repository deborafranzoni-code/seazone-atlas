# rating

Descrição: Um número de ponto flutuante que representa a classificação da propriedade agregada. Normalmente, esse número vai de 0 a 5, de 0 a 10 ou de 0 a 100, mas é possível usar qualquer intervalo que represente seu sistema de classificação. Incluso no XML?: Sim Obrigatório?: Strongly recommended Observação: Apenas enviado se a propriedade tiver algum rating (Comentário) Valores de conteúdo válidos: Non-negative float value
