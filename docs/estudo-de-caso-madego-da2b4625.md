# Estudo de Caso Madego


\
**DESCRIÇÃO**

ℹ️

O objetivo desse estudo é entender e compreender todas as situações que trouxeram dificuldades as equipes durante o processo de implantação e buscar soluções para que em uma nova oportunidade, o processo ocorra de uma maneira mais simplificada e assertiva.


**PROCESSO**

* Mapeamento dos Impactos

  Foi realizado um mapeamento com todas as informações relevantes para observar os impactos, entender o que gerou cada um deles e buscar uma solução para que não ocorram novamente ou que ocorram de uma maneira mais sutil e que possibilite uma fácil resolução.

  Link para visualização do mapeamento: [Estudo de Caso Madego, Visual Workspace for Innovation (miro.com)](https://miro.com/app/board/uXjVKfBbowA=/#tpicker-content)
