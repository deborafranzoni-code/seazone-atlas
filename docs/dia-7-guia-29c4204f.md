# Dia 7 - GUIA

**O que será feito?**

* Acompanhamento na implementação de alguma task de **NÍVEL INICIANTE / INTERMEDIÁRIO** que seja necessário documentar e implementar testes automatizados utilizando o Cypress

**Objetivo do treinamento**

* O objetivo desse treinamento é, além de reforçar todos os conceitos já vistos nos dias anteriores, implementar na prática o primeiro teste automatizado em conjunto com a pessoa a quem este treinamento se direciona

**Exemplo de tasks de NÍVEL INICIANTE / INTERMEDIÁRIO que poderiam ser repassadas:**

* ***Editar username, telefone ou qualquer outra informação que seja relativamente simples de alterar e que necessite fazer requisição a API***
* ***Correções mais simples***
* ***Integração de dados mockados de páginas/modais mais simples***
