# Atualizar dados bancários

Quando atualizar  dados bancários seja de Proprietários ou Franqueados via "Suporte Operacional"  deve-se atualizar também os seguintes campos associado ao UserID do Proprietário ou Franqueado

Tabela: 

`Financial bank Details > is Account Validated = False`

`Financial bank Details > is Pix Key Validated = False`


O `UserID` é do Proprietário ou Franqueado
