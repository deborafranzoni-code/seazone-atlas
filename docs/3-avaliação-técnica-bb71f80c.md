# 3. Avaliação Técnica

## Objetivo

O que se deseja entregar e seu valor para empresa.

## Viabilidade

No ambiente de time atual, temos recursos e conhecimento técnico para realizar a entrega?

## Escopo

O que é e o que não entra na entrega

## Complexidade

### Tecnologia

Tecnologias envolvidas, linguagens, ambientes de cloud

### Integrações

A entrega acontece de forma independente ou precisa se comunicar com outros sistemas já em produção?

### Legado

Precisamos utilizar/entregar sistemas legados para realizar o épico?

### Time

Quantidade de membros do time necessários para realizar a entrega Precisamos usar sistemas legados para realizar a entrega?

### Documentação

Temos documentação sobre a entrega, planilhas, diagramas, descrições do produto. Documentação clara das ferramentas/processos que serão implementados

## Dealine

Prazo de entrega

## Definition of Done (DoD)

Como cliente, delimitação de detalhes do que se deseja entregar.
