# No-Show


\
Sempre que um hóspede cancela a reserva fora do prazo limite para cancelamento, é cobrada uma multa pelo cancelamento. Essa multa nós chamamos de no-show (não comparecimento).

* Como verificar um pagamento de no-show

Os valores recebidos por no-show deverão ser transportados da planilha de conciliação para a [planilha de no-show](https://docs.google.com/spreadsheets/d/1mcXhCIeYReeYTVStWKxTUQE6gqG9WTjYvhy3Td6Y688/edit#gid=400624681)

* Como passar as informações para a planilha de no-show

Depois de transportar os dados para a planilha, os recebimentos de no-show deverão ser categorizados da seguinte forma:

* Repasse ao proprietário
* Fica com szs
* Estorno Integral
* Danos de hóspede

## Objetivo

Identificar e categorizar os pagamentos recebidos por no-show

**Semanal**
