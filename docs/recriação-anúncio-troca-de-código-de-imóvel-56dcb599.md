# Recriação anúncio - Troca de código de imóvel

Este processo deve ser seguido sempre que um imóvel trocar de código na Stays.\n

# Troca de código com um novo calendário na STAYS


1. Inative o imóvel na tela `Operacional >> Implantação >> Mudança de Status`

 ![](/api/attachments.redirect?id=bc3c55ce-1d9c-4b7b-820d-fd8a811fe5b5 " =1867x841")


:::warning
A data de inativação do imóvel não deve coincidir com nenhuma reserva vigente, caso isso aconteça, esta reserva NÃO será migrada da Stays e nem considerada para fins de fechamento

:::


2. Cadastre o novo código de imóvel na tela `Operacional >> Implantação >> Onboarding`\na) Selecione o proprietário do imóvel\nb) Clique no botão "Adicionar imóvel"\n\n![](/api/attachments.redirect?id=3b9fa310-86ea-4d8e-9a4d-4d5715f417f8 " =1914x578")

   \nc) Preencha a sessão de "`Dados do imóvel`" e inclua o Código do imóvel ==exatamente== como está na Stays.

   
:::warning
   Caso o código do imóvel não esteja preenchido exatamente como está na Stays, o calendário do imóvel não será sincronizado

   :::

   \nd) Na sessão "`Taxas e comissões`" selecione os dados de contrato do imóvel, considerando que a ==Taxa de adesão deve estar zerada==\n\ne) Selecione o Anfitrião responsável pelo imóvel.

   
:::warning
   O anfitrião vem selecionado por padrão como "A DEFINIR". Por isso ele deve ser selecionado neste momento para que não seja necessário realizar a troca de anfitrião.

   :::

   f) Finalize o formulário preenchendo todas as informações restantes e clique em salvar.

   
:::warning
   Caso alguma reserva do calendário antigo seja migrada para o calendário novo é necessário ajustar a data de início do contrato na tela de "Operacional >> Implantação >> Editar Propriedade". A data de início de contrato deve ser igual a data de check-in da reserva. \n\nCaso não seja possível selecionar a data no Sapron, abra um suporte para ajuste.\n\nIsto deve ser feito para garantir que a reserva fique no calendário certo do imóvel e seja considerada para fins de fechamento

   :::
3. Verifique que todas as informações preenchidas estão corretas imediatamente após o preenchimento dos formulários, na tela de Multicalendário, e que todas as reservas constam no  imóvel novo.
4. Ative o imóvel na tela de Mudança de Status. Garanta que a data de ativação seja igual ou menor as datas das reservas vigentes.\nGaranta que a data de início do contrato do imóvel (tela de editar propriedades) é igual a data de ativação do imóvel
