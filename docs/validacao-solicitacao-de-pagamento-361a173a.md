# Validação - Solicitação de Pagamento


**Processo de Validação**


Através da solicitação de pagamento enviada via [formulário](https://docs.google.com/forms/d/e/1FAIpQLSeb_g5gSfNSp5s1cvtgBojn0_9Nlm30E5HgFC6p7-1MbyefIw/viewform) pelos colaboradores, a sua solicitação para pagamento irá diretamente para a página de controle do financeiro ([PGTO_REQ.](https://docs.google.com/spreadsheets/d/12jFzCTYAISO2xVBFcC6YkDehBmR2m5ymYGcwQnU3fhA/edit#gid=1047151544)). Entretanto, o pagamento só será efetuado após a aprovação do diretor responsável.\n\n\n\n**Modelo de Aprovação**


Após preenchido formulário, a solicitação de pagamento irá aparecer na planilha do diretor responsável, contendo as informações, como:\n- ID da solicitação;\n- Diretor responsável;\n- Solicitante do pagamento;\n- Empresa do gasto;\n- Setor;\n- Centro de Custos;\n- Motivo;\n- Valor;\n- Forma de Pagamento;\n- Data de Vencimento;\n- Link Nota Fiscal.\n\nAlém destes campos, irá conter duas caixas para seleção:\n\n- Aprovado: deverá ser selecionada quando autorizada para pagamento;\n- Reprovado: deverá ser selecionada quando negada para pagamento.\n\n\nAssim que marcada a box, esta resposta será enviada automaticamente para o financeiro resultando no processo retornado pelo diretor do setor.\n\n
