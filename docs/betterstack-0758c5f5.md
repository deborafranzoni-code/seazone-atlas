# BetterStack

[Better Stack: Spot, Resolve, and Prevent Downtime.](https://betterstack.com/)

### Serviços utilizados

* **[Logtail](https://betterstack.com/logs)**
  * Ferramenta de Log Drain.
  * Dispara alertas no slack no canal \*\*`#website-alerts**` quando ocorrer erros: 400, 404, 403, 421, 500 e 504
* **[Uptime](https://betterstack.com/uptime)**
  * Status Page

  [Seazone status](https://seazone-reservas.betteruptime.com/)
  * Alertas de indisponibilidade
