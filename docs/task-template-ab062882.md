# Task Template

|    |    |
|:---|:---|
| Do Date | @September 29, 2023 |
| Task recorrente? |    |
| Created time | @September 21, 2024 7:00 AM |
| Do date formula | September 29, 2023 |
| Periodicidade |    |
| Multi-select |    |

IF the task is not recurring, the formula and the do date property is going to be the same

IF the task is recurring, the do date is going to be the task creation date plus the delay

IF the task is recurring and I add a do date to it, that do date is going to override the previous date. (Flexibility to reschedule).
