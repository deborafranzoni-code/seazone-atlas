# Mapa de Terrenos

O Mapa de Terrenos é um site desenvolvido para facilitar a visualização, filtragem e análise de terrenos por meio de um mapa interativo. A plataforma permite que o usuário localize terrenos, consulte informações detalhadas, aplique filtros personalizados e visualize diferentes camadas de dados geográficos.\n
