# 🖥️ Onboarding

## 🎉 Boas-vindas!

Seja muito bem-vind@ ao time de Hosting da Seazone! Aqui, nosso foco é desenvolver e manter features que apoiam dois segmentos fundamentais para o negócio: os **proprietários (investidores)** e os **anfitriões (franquias)**. Para isso, trabalhamos principalmente em duas ferramentas: a **Wallet** e o **Sapron**, que você vai conhecer melhor ao longo desta documentação.

Este documento foi pensado para te guiar nos primeiros passos, de forma simples e prática. A ideia é que você encontre aqui o essencial para começar bem: desde onde falar com o time até como configurar seu ambiente de desenvolvimento. 

Bora lá? :eyes:
