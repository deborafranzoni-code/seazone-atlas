# Edição dos links do menu (seção seazone serviços)

* **Como alterar um link do menu?**
  * No prismic clique na página `Menu`;
  * Procure o link que deseja alterar;
  * No campo `*url*` apague a url antiga e digite a nova url;
  * Clique no botão `Salvar`;
  * Em seguida para que a alteração seja publicada em produção clique no botão `Publish`.
* **Como alterar o título do link no menu?**
  * No prismic clique na página `Menu`;
  * Procure o título do link que deseja alterar;
  * No campo `*Título do link*` apague o título antigo e digite o novo título;
  * Clique no botão `Salvar`;
  * Em seguida para que a alteração seja publicada em produção clique no botão `Publish`.
