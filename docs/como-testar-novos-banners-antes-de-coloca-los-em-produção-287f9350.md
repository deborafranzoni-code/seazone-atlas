# Como testar novos banners antes de coloca-los em produção ?

Links uteis:

* [Pismic de desenvolvimento](https://seazone-reservas-development-cms.prismic.io/documents/working/)
* [Staging do website](https://web-stg.seazone.com.br/)


\
É possível realizar testes nos banners antes que sejam incluídos no website principal da seazone

Isso é feito através do prismic de desenvolvimento 

Basta acessar o prismic de desenvolvimento pela aba de links uteis acima.

O prismic de desenvolvimento possúi as mesmas configurações que o de produção, a unica diferença é que todas as imagens adicionadas no de desenvolvimento terão efeito apenas na staging do website, cujo link também esta acima, então sinta-se livre para adicionar imagens e realizar testes nesse prismic sem medo.


Importante mencionar que o website utiliza features flag para mostrar informações em vários lugares, pode ser que para mostrar o seu banner seja necessário habilitar uma flag.
