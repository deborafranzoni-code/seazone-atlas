# Sanity Check Sheets x Sapron

**DESCRIÇÃO**


ℹ️

Essa planilha tem o objetivo de comparar o fechamento entre o SAPRON e o sheets.


**PRAZO**

⏰

Após concluir o fechamento de Imóvel, Proprietário e Anfitrião pela planilha.

\n\n<https://docs.google.com/spreadsheets/d/1TxUPzQfoDuWA5AzUbO-L_hkJEN-ouq7l-wvi82n-L1I/edit#gid=2142696512>\n\n

**ALTERAÇÃO TEMPLATE DE SANITY CHECK**

* **Passo a passo (para os planos A)**

  
  1. Acessar a planilha [Sanity Check: SAPRON <> FECHAMENTO](https://docs.google.com/spreadsheets/d/1aEhxT50U3AyqLMbC8jicaZUsvC0EaNSge-vdKTO92BI/edit?gid=699961851#gid=699961851) na aba Dash;
  2. Atualizar o ID Fechamento, Ano de Referência e Mês de Referência;
  3. Deve-se selecionar, na aba Dash, a base de acordo com o que deseja conferir (Proprietário, Anfitrião ou Imóvel).
* **Passo a passo (para os planos B)**

  
  1. Acessar a planilha [Legacy => Sanity Check: SAPRON <> FECHAMENTO](https://docs.google.com/spreadsheets/d/1ywvLiGLOB9nCGjGcFuiKR2IqTr6t1k-P7F_niX_Bkl0/edit#gid=0) na aba Dash;
  2. Atualizar o ID Fechamento, Ano de Referência e Mês de Referência;
  3. Deve-se selecionar, na aba Dash, a base de acordo com o que deseja conferir (Proprietário, Anfitrião ou Imóvel).


**IMÓVEL**

* **Passo a passo (PLANO A)**

  
  1. Acessar a planilha [Sanity Check: SAPRON <> FECHAMENTO](https://docs.google.com/spreadsheets/d/1aEhxT50U3AyqLMbC8jicaZUsvC0EaNSge-vdKTO92BI/edit?gid=699961851#gid=699961851) na aba Dash
  2. Ir no menu superior, clicar em Atualizar → Atualizar Sapron → Imóvel *(ou Atualizar Sapron → Rodar Tudo)*;
  3. Verificar as divergências entre a planilha [3.0 BizOps - Fechamento Mensal Template - BizOps](https://docs.google.com/spreadsheets/d/1o216mX4uUnUAwFhUi3pfFtTdg2wlSEmo6Eh08gdTu50/edit#gid=1073825828) e o [Sapron](https://sapron.com.br/login) apontadas na aba Dash;
  4. Utilizar a aba Imóvel da planilha [Sanity Check: SAPRON <> FECHAMENTO](https://docs.google.com/spreadsheets/d/1aEhxT50U3AyqLMbC8jicaZUsvC0EaNSge-vdKTO92BI/edit?gid=699961851#gid=699961851) para consulta, se necessário;
  5. Solucionar as divergências caso a caso;
  6. Se houverem casos em que a divergência pode ser desconsiderada, anotar na aba Imóvel (mês de referência) da planilha [Conferencia Sanity Check](https://docs.google.com/spreadsheets/d/1rhrqIgtRT-LeEl8NRUvFiSDEGZsD34A1Y6kxbnH3STc/edit?gid=511777784#gid=511777784) para consulta posterior.
  7. Ao final da conferencia clicar no menu superior, clicar em envio→ All, para enviar todos os dados para o [Compilado de problemas de Fechamento](https://docs.google.com/spreadsheets/d/1wODwZthfQE4yU3FcWUHPpOEAMa7yDgdlU64QTiWaCKk/edit?gid=0#gid=0).
* **Passo a passo (PLANO B)**

  
  1. Acessar a planilha [Legacy => Sanity Check: SAPRON <> FECHAMENTO](https://docs.google.com/spreadsheets/d/1ywvLiGLOB9nCGjGcFuiKR2IqTr6t1k-P7F_niX_Bkl0/edit#gid=0) na aba Dash
  2. Ir no menu superior, clicar em Atualizar → Atualizar Sapron → Imóvel *(ou Atualizar Sapron → Rodar Tudo)*;
  3. Verificar as divergências entre a planilha [00 - Fechamento Mensal Template](https://docs.google.com/spreadsheets/d/1UoGdsSrhNuyeTupWzP3tgg1mKUF81X8Ac6rwspoTRD0/edit#gid=1073825828) e o [Sapron](https://sapron.com.br/login) apontadas na aba Dash;
  4. Utilizar a aba Imóvel da planilha [Legacy => Sanity Check: SAPRON <> FECHAMENTO](https://docs.google.com/spreadsheets/d/1ywvLiGLOB9nCGjGcFuiKR2IqTr6t1k-P7F_niX_Bkl0/edit#gid=1213685698) para consulta, se necessário;
  5. Solucionar as divergências caso a caso;
  6. Se houverem casos em que a divergência pode ser desconsiderada, anotar na aba Imóvel (mês de referência) da planilha [Ignorar no Sanity Check](https://docs.google.com/spreadsheets/d/1R8X1NsmSqm6XknMUCIoFDV-5Ol9fhnvCy-m07YMDozQ/edit#gid=1006654936) para consulta posterior.
