# Projeto criação de anúncios automatizados

1. **Introdução**

* **Contexto:** Atualmente a Seazone utiliza a **Stays** para gerir os anúncios nas OTAs: Airbnb, Booking, Expedia, Decolar e Website Seazone.\nA partir do momento em que um imóvel é fechado pelo time comercial, o imóvel passa por uma vistoria (utilizando a ferramenta Pleno). Se o imóvel estiver apto para locação ele então é criado na Stays, e na sequência, em todas as OTAs mencionadas.\nTodo este processo se dá de forma manual, levando cerca de 2 horas para criar um único anúncio. \nO que significa que para criar 10 anúncios, levamos 20h, ou quase 3 dias de trabalho de um colaborador da operação.\n\nA documentação completa de criação de anúncios pode ser encontrada aqui:\n[Criação do anúncio base Stays](https://app.slack.com/client/TDLTVAWQ6/dms#:\~:text=o%20seguinte%20canvas%3A-,Cria%C3%A7%C3%A3o%20do%20an%C3%BAncio%20base%3A%20Stays,-Fabi%20Azevedo)\n[Anúncios](https://seazone-fund.slack.com/docs/TDLTVAWQ6/F0873CKS11C)
