# REC_TAXAADM_POOL


**Descrição**

É uma prestação de serviços com as seguintes atividades.

* Auditoria e categorização de contas;
* Elaboração de demonstrativos de rentabilidade;
* Recebimento e repasse dos pagamentos de locações;
* Manutenção do imóvel.


**Quando e como serão os recebimentos do Serviço?**

O Responsável enviará o contrato fechado para o financeiro. No contrato constará o valor total e forma de pagamento, bem como os dados para emissão da nota fiscal.\nSerá pago via mensalidade entre dias 20 e 25.\nEnviar a nota fiscal no começo de cada mês.\n


**Regime Tributário e como emitir a nota Fiscal de Comissão Seazone Investimentos.**

* Regime Tributário = Lucro Real
* CST = 0 (Tributada Integralmente).
* Regime Especial de Tributação = Automático (utilizado para plataformas)
* Valor aproximado dos Tributos (IBPT) = 16,45%
* Impostos IRRF 1.5% e CRF 4,65% (colocar na parte de informações adicionais.)
* Cnae = 6822600 (**Gestão e administração da propriedade imobiliária**)
* Descrição do Serviço = Taxa Adm Pool\nDados bancários da SZI: SEAZONE INVESTIMENTOS LTDA CNPJ (PIX): 34.226.198/0001-48) Conta Inter 77 - Banco Inter S.A Ag: 0001 CC: 3570900-\n
* Modelo da Nota fiscal.

* **Sistema Prefeitura (via site)**

O Sistema da Prefeitura disponível via site é utilizado quando a emissão de Notas Fiscais ocorre individualmente. São casos esporádicos conforme a demanda, sem data definida para a emissão mensal.
