# Disparos de notificação para contatos

# Objetivo Geral : 

Garantir que a cadência de disparos de notificação para contatos seja respeitada, variando de acordo com o formato da campanha : 

* **Whatsapp :** O contato só está elegível para receber esta forma de comunicação 3 dias após o último recebimento
* **E-mail :** O contato só está elegível para receber esta forma de comunicação 1 dia após o último recebimento

  \
  *\*\*\* Observação : Comunicados informativos não respeitam a regra de 3 dias da comunicação via Whatsapp, mas bloqueiam outras comunicações que poderiam ocorrer (E-mail)*

# Fluxo de Operação Marketing : 

Atualmente, o fluxo operacional do marketing divide-se em 4 etapas : 


1. Receber a solicitação do sabor via [Pipefy](https://app.pipefy.com/pipes/306792041/form)  (o usuário que solicitou  envia a base de contatos)
2. O time analisa a demanda, baseando-se na data que o usuário gostaria de enviar o disparo 
3. As informações são válidadas através do cruzamento de dados com uma [planilha de disparos](https://docs.google.com/spreadsheets/d/1G3IkbBwYFlPh2abttFQS4JzhCr9PG3iO_QfTHqymE2Y/edit?gid=0#gid=0)
4. Base é destinada ao time de Marketing, que encaminhará a  plataformas que farão o envio (MIA, NewByte, RD Station, etc)


# Objetivo do Projeto :

Garantir que haja uma estrutura escalável e consolidada para a geração de bases de contatos, dos mais variados sabores (tipos de pedidos dos usuários)


# Sabores : 

Os sabores são entendidos como o tipo de base de contatos que um usuário pode solicitar :



| Vertical | Descrição |
|----|----|
| Clientes SZS  | Atuais proprietários de  cada imóvel contidos no SAPRON |
| Hóspedes  | Pessoas que já se hospedaram em imóveis gerenciados pela SZN |
| Clientes - SZI/MKTP | Atuais proprietários de cada unidade de SPOTS |
| Lost - Pipedrive | Traz informações sobre os deals perdidos do Pipedrive (CRM) |
| Leads - RD Station | Contatos que já converteram em alguma campanha de marketing |
| Parceiros | Parceiros da SZN contidos no SAPRON |
| Churn | Proprietários de Imóveis que deixaram de utilizar o serviço da SZN |
