# Google Cloud

## Serviços utilizados

* Google Maps API
* OAuth2 configurado para logar com o Google no [Auth0](/doc/auth0-PsX5OvzmUU)

## Como obter acesso à conta

Acesse o console do Google Cloud: [https://console.cloud.google.com](https://console.cloud.google.com/), e faça login usando uma conta do Google que tenha acesso ao projeto **site-de-reservas** no Google Cloud.

A conta principal (dona do projeto) é a `sitedereservas@seazone.com.br` .

Se deseja ter acesso ao projeto no Google Cloud, fale com o Bernardo Ribeiro no Slack.

**Link direto para o projeto (é preciso ter sido adicionado ao projeto):** [console.cloud.google.com/welcome?project=site-de-reservas&supportedpurview=project](https://console.cloud.google.com/welcome?project=site-de-reservas&supportedpurview=project)
