# Dados Retroativos - Reservas

# Mudar Taxa de Limpeza para a reserva extensão

Quando por algum motivo de ajuste de dados é solicitado mudar a taxa de limpeza da `reserva original` para a `reserva extensão`

Precisa mudar os campos

* `Cleaning Fee Value`
* **Net Cleaning Fee**

O fechamento considera principalmente o campo `Net Cleaning Fee`


Exemplo, neste [suporte](https://app.pipefy.com/open-cards/1140157906). Foi solicitado migrar a taxa de limpeza da reserva original para a reserva da extensão2
