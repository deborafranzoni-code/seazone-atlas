# Fluxo de Operação

# Desenvolvimento do projeto : 

A solução foi pensada em etapas conforme ilustra o diagrama, com as seguintes etapas : 


1. Construção das tabelas de contatos e seus parâmetros
2. Consolidação das Querys de cada sabor
3. Criação de uma única tabela de contatos
4. Construir a tabela de disparos em um banco de dados (sem depender de uma planilha)
5. Automatizar o envio da base para o solicitante
