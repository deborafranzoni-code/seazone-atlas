# Dia 4 - GUIA

**O que será feito?**

* Acompanhamento na implementação de alguma task de **NÍVEL INICIANTE** que não seja necessário realizar integração com api justamente para facilitar o entendimento da pessoa a quem este treinamento se direciona

**Objetivo do treinamento**

* O objetivo desse treinamento é revisar na prática o processo de implementação de uma nova task desde o princípio a fim de reforçar a fixação dos processos de: *criação de uma nova branch*, *documentação*, *commits* e *preenchimento do template da PR*

**Exemplo de tasks de NÍVEL INICIANTE que poderiam ser repassadas:**

* ***Correções de interface e bugs mais simples***
  * ***Alterar a cor de um botão***
  * ***Adicionar um hover em algum componente***
  * ***Ocultar algum componente de alguma tela do Sapron***
  * ***Formatar números ou strings (ex: aplicar máscara de CPF, aplicar máscara de dinheiro)***
