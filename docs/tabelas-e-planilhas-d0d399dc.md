# Tabelas e Planilhas

Esta seção tem como finalidade ser um dicionário de dados de cada tabela usada por nosso time, ou seja, uma explicação de forma sucinta sobre o que é possível encontrar em cada.

* **Tabela de Contas/Login**

  *Account User* é a tabela onde armazena todos os usuários do Sapron (sendo parceiros ou não). Nela fica as informações que todos os usuários possuem, como o nome, cpf, e-mail, etc.
* **Tabela Conta de Parceiros**

  *Account Partner* é a tabela utilizada para armazenar informações especificas sobre nossos parceiros, como: nacionalidade, status civil, profissão, executivo responsável entre outros. É uma "especialização" ou "herança" da tabela de usuários
* **Tabela de comissões de Terrenos**

  *Financial Partners Commissions Allotment* é a tabela onde contém registros sobre as comissões de indicações de terrenos. Ela possui alguns dados sobre nossos parceiros como: ID, Valor, Status, Data, entre outros.
* **Tabela de comissões de imóveis**

  *Financial Partners Commissions Reservation:* Esta tabela contém registros sobre as comissões de indicações de reservas (imóveis), e ela possui alguns dados como: ID, valor, status, indicações, etc.
* **Tabela de comissões pagas de parceiros**

  *Financial partners commissions withdraw paid*\*\*:\*\* Esta tabela é uma espécie de carteira virtual dos nossos parceiros, ela possui registros sobre os resgates (saques) realizados e recebidos pelos parceiros.
* **Tabela de comissões solicitadas pelos parceiros**

  *Financial Partners Commissions Withdraw Request*  é a tabela onde contém os registros sobre as solicitações de resgates (saques) realizados e mas que ainda não foram recebidos pelos  parceiros.
* **Tabela de Indicações de Terrenos**

  *Partners Indications Allotment* é a tabela que armazena dados como os registros das indicações de terrenos realizadas pelos nossos parceiros. Ela possui dados como ID, nome e telefone do proprietário do terreno, status, etc.
* **Tabela de Indicações de Imóveis**

  *Partners Indications Property* é a tabela onde contém registros das indicações de imóveis realizados pelo parceiro, ela possui dados como ID, status, Data, Cidade do propriedade indicada, Contatos do Proprietário, Comentários, etc.
* **Tabela Propriedade**

  *Property_Property* é a tabela onde contém o registro de todos os imóveis em nosso banco de dados.
