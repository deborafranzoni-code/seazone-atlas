# Dashboard de Auditoria de Variação de Concorrentes

Dashboard desenvolvido para o time de Revenue Management (RM) com o objetivo de auditar variações no número de concorrentes entre duas datas. Além de mostrar KPIs de ganhos, perdas e variação líquida, ele detalha o motivo de cada mudança, permitindo um diagnóstico rápido.
