# Produtos & Serviços

PIPE

> **Nos referimos ao nosso pipeline de dados externos como o "Pipe".**
>
> O Pipe é composto por:
>
> * scrapers, responsáveis pela aquisição de dados
> * lake, responsável pelo armazenamento e organização dos dados
> * jobs, responsáveis pelo processamento (limpeza e enriquecimento) dos dados
> * modelos, responsáveis pelo entendimento
