# Métricas das páginas

Para realizar a análise das métricas usamos o site **[PageSpeed Insights](https://pagespeed.web.dev/)** para obter o valor das métricas de **Desempenho, Acessibilidade, Práticas Recomendadas e SEO.**

Após a realização do teste através do **PageSpeed,** salvamos o resultado na planilha de **[PageSpeed Insights - Sites de Reservas](https://docs.google.com/spreadsheets/d/1DMxtqsFKLxs9y7PesVkCS1WupKd6JaJ9sfy2RFhl5ZQ/edit#gid=630644247)** que contém todo o histórico das métricas.

Atualmente, a frequência dessa verificação é semanal.

É esperado que:

* **Desempenho ≥ 50**
* **Acessibilidade ≥ 90**
* **Práticas Recomendadas ≥ 90**
* **SEO ≥ 90**
