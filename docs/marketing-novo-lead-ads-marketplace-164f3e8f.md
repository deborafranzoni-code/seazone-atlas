# Marketing - Novo Lead Ads (Marketplace)

![](/api/attachments.redirect?id=48dbf7e2-2a79-42f5-ad8b-9911d8cec508 " =1100x550")

A migração do [Make.com](http://Make.com) para n8n visa resolver problemas estruturais da arquitetura anterior:

* **Eliminar duplicação de código**: Substituir cenário problemático por um workflow parametrizável, reduzindo a superfície de manutenção
* **Melhorar governança**: Implementar tratamento de erros robusto e centralizado, com notificações estruturadas via Slack para visibilidade total do time
* **Facilitar manutenção**: Externalizar todas as configurações de campanha para o Baserow, permitindo alterações de regras de negócio sem necessidade de deploy ou alteração de código
