# Paypal

## O que é?


---

O Paypal é um dos meios de pagamento utilizado no Site de Reservas. Através dele que conseguimos criar os checkouts e receber os pagamentos das reservas que são criadas pelo Site de Reservas.

Diferente do PagarMe, o Paypal é utilizado principalmente para o receber o pagamento de reservas de hóspedes internacionais que desejam pagar em uma moeda diferente do Real brasileiro.

## Para que usamos?


---

Atualmente usamos o Paypal como Gateway de pagamento para pagamentos internacionais (mas é possível receber de hóspedes nacionais) por meio de: Cartão de Crédito (À vista)

Realizamos a comunicação com o Paypal utilizando as APIs\*\*,\*\* para gerar um pedido (link para o checkout) bem como integrações com Webhook para receber atualizações sobre os pagamentos (se ele foi pago, cancelado, estornado, falho, etc).

Para pagamentos nacionais utilizamos o [Pagar.Me](/doc/pagarme-ASiptJ5gJR)

## Como acessar?


---

Necessário requisitar um acesso ao seu Gestor.
