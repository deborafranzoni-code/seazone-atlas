# Validação Sapron ✅


\

**Descrição**


Os Imóveis devem ser validados no Sapron após a aplicação do Sanity Check com o Sheets e de sanadas as possíveis divergências.


\
**Lembrete Importante**


⚠️⚠️ A Lista de TEDs do Sapron só poderá ser gerada após a validação dos imóveis ⚠️⚠️


\
**Processo**


* Acessar o Sapron

[Sapron](https://sapron.com.br/)


* Acessar "Fechamento Imóvel" na barra de tarefas lateral


* Selecionar o mês de referência para o Fechamento


* Selecionar todos os imóveis


* Clicar no botão "Ações" situado no canto superior da tela


* Clicar no botão "Validar Dados"


* Conferir se todos os imóveis foram validados digitando "Open" na barra de pesquisa (caso algum imóvel apareça é preciso validar individualmente)


* **Carolina Espindola e Gladys** devem ser avisadas via canal "Operação" do Slack sobre a validação.


\
**Observação**


A validação de Proprietários e Anfitriões deve ser realizada apenas após 3 meses de Fechamento sem erros no Sapron
