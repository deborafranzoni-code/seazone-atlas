# Edição dos links do rodapé

* **Como alterar um link do rodapé?**
  * No prismic clique na página `Rodapé`;
  * Procure o link que deseja alterar;
  * No campo `*url*` apague a url antiga e digite a nova url;
  * Clique no botão `Salvar`;
  * Em seguida para que a alteração seja publicada em produção clique no botão `Publish`.
* **Como alterar o texto do link no rodapé?**
  * No prismic clique na página `Rodapé`;
  * Procure o Texto do link que deseja alterar;
  * No campo `*contato*` apague o texto antigo e digite o novo texto;
  * Clique no botão `Salvar`;
  * Em seguida para que a alteração seja publicada em produção clique no botão `Publish`.
* **Como alterar o título do link no rodapé?**
  * No prismic clique na página `Rodapé`;
  * Procure o título do link que deseja alterar;
  * No campo `*Título do link*` apague o título antigo e digite o novo título;
  * Clique no botão `Salvar`;
  * Em seguida para que a alteração seja publicada em produção clique no botão `Publish`.
* **Qual a diferença entre título do link e texto do link?**
  * ***Título do link*** é o que aparece em cima do link e ***texto do link*** é o texto que faz referência ao link(geralmente é algum contato ou informação importante).
    * Ex: **Quero anunciar meu imóvel** é o título do link, já o **+55 (48) 2500-0646** é o texto do link.
