# Kit boas-vindas

Toda segunda feira é verificado os novos Seazoners e preparado Kit para envio via transportadora.\n\n

## Relatório de novos Seazoners;

Retirar relatório do [Convênia](https://app.convenia.com.br/insights/relatorios/pre-definidos) de novos Seazoners  com login de administrador, alimentar a planilha ( [Relação Colaboradores para envio de kits](https://docs.google.com/spreadsheets/d/1AN-cgcDIaNW8I87gqg0dw950g2XsHMSl5FXCktz6mxM/edit?usp=sharing) ) com os novos, confirmar endereço pelo slack ( <https://app.slack.com/client/TDLTVAWQ6/CDKD79F6F>  ) com o Seazoner, atualizar a planilha se necessário;

## Separação e montagem dos Kits para envio;

Preparação das embalagens com papel kraft ou caixa dependendo do tamanho e dos itens, montagem de acordo com os itens;

Sempre com o bilhete de boas vindas ou de novo Mimo;

Emitir Declaração de Conteúdo, LOG de envio e Romaneio na planilha ( [Planilha para envios do Administrativo](https://docs.google.com/spreadsheets/d/1DQmv1bHXuUxZWSZDdrtdGS2evUra57qQF_0HMqrKf7U/edit?usp=sharing) ), salvar os Logs Consolidados no bloco de notas (Salvar arquivo em .cvs), acessar o site da transportadora [Total](https://ics.totalexpress.com.br/) , a senha de acesso está no Vault, fazer o [upload dos dados](https://ics.totalexpress.com.br/remetentes/v3/coletas_upload.php) , todos os envios para o dia , [listar coletas](https://ics.totalexpress.com.br/remetentes/coletas.php), imprimir as etiquetas da coleta (Etiqueta Long Tail (A4 - Pedido));

Colar nas embalagens de maneira visível a etiqueta da transportadora e declaração de conteúdo;


## Coleta pela transportadora;

Avisar no dia anterior a coleta quantos volumes a transportadora virá buscar (Via WhatsApp +55 48 9912-6526 ) ;

Sempre entregar mediante assinatura do romaneio e conferência da transportadora com assinatura;\n\n\n

## Controle do recebimento via site;

Conferência via site da transportadora para atualizar o Log de Coleta > enviado> entregue ( <https://ics.totalexpress.com.br/remetentes/v3/search.php> )

Sempre verificar inconsistências na entrega serão informados no site;


## Recebimento do Kit pelo Seazoner;

Tempo médio do Transporte: 04 à 12 dias.


# Itens do kit - 2024

* Garrafinha;
* Caderno;
* Camiseta;
* Copo Seazone (quando disponível);
