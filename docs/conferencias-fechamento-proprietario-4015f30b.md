# Conferências Fechamento Proprietário


\
**Descrição**


São métricas de conferências aplicadas durante o processo de Fechamento para precaver erros e trazer mais segurança nos dados apontados no Fechamento de Proprietários.


Proprietário

O que é verificado?

* Saldos e Taxas de Implantação dos imóveis
* Receita
* Despesas
* Comissão
* Repasse
