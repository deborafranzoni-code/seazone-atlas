# Problemas Conhecidos

## Descrição:

Registros de falhas recorrentes, limitações técnicas de ferramentas ou comportamentos esperados que ainda não possuem solução definitiva. Esses problemas podem impactar entregas, criar confusão na operação ou exigir ações manuais frequentes. Quando aplicável, o documento deve conter soluções temporárias (workarounds) para mitigar os efeitos.
