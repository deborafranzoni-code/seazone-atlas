# Pagar.Me

## O que é?


---

O PagarMe é um [Gateway de pagamento](https://pagarme.helpjuice.com/pt_BR/sobre-o-pagarme/o-que-%C3%A9-gateway) e também um [PSP (Payment Servide Provider)](https://pagarme.helpjuice.com/pt_BR/sobre-o-pagarme/o-que-%C3%A9-psp). É através dele que conseguimos criar os checkouts e receber os pagamentos das reservas que são criadas pelo Site de Reservas.

## Para que usamos?


---

Atualmente usamos o Pagar.me como Gateway de pagamento para pagamentos nacionais por meio de: Cartão de Crédito (À vista, Parcelado) ou PIX.

Realizamos a comunicação com o Pagar.me utilizando as APIs do **Dashboard v5,** para gerar um pedido (link para o checkout) bem como integrações com Webhook para receber atualizações sobre os pagamentos (se ele foi pago, cancelado, estornado, falho, etc).

Para pagamentos internacionais utilizamos o [Paypal](/doc/paypal-ZtCsdkMy9Q).

## Como acessar?


---

Necessário requisitar um acesso ao seu Gestor.
