# Investigação de suportes

## Comissões

Exemplo: Se a propriedade tem comissão de 20%, quer dizer que o proprietário paga 20% para seazone(szn),  o total pode ser dividido da seguinte forma:

* 15% - host 
* 5% - szn

As porcentagens da comissão podem ser variáveis (subir/baixar).


Para verificar as valores da comissão pesquise os dados atuais nas seguintes tabelas

* **property_property**: obter o `code` ou `property_id`
  * coluna `comission fee`
  * <https://metabase.sapron.com.br/question/1089-comision-property>

  \
* **property_audit** :  estão todas as comissões das propriedades
  * `Comission Fee`
  * `Host Reservation Comission Fee`
  * <https://metabase.sapron.com.br/question/1087-comision-property-audit-by-propertycode>

    \
* **reservation_reservation**:  estão todas as reservas e suas respectivas comissões, deve considerar a coluna created_at 
  * \
  * <https://metabase.sapron.com.br/question/1088-comision-reservation-by-property-id>
