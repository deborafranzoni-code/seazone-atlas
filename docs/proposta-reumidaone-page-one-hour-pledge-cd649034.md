# Proposta reumida(One Page/ One Hour Pledge)

## **CONCEITO**

Desenvolver um sistema de geração de relatórios acionáveis usando IA Generativa para análise de desempenho de imóveis, baseado em features e parâmetros validados através de análise exploratória de dados históricos.

## **METODOLOGIA**

### **Fase 1: Análise Exploratória (Julho/2025)**

* Análise de 16.564 registros com snapshots diários
* Identificação de padrões de desempenho e definição de features críticas
* Desenvolvimento de 8 features validadas estatisticamente
* Criação de funil de classificação em 3 grupos principais:
  * Quase Lá (85-99% meta) - *foco inicial*
  * Overperformers (≥200% meta) - *performance excepcional*
  * Críticos (<85% meta) - *baixo desempenho*

### **Fase 2: Sistema de Diagnóstico**

* Implementação de regras de negócio e features calculadas
* Enriquecimento de dados com informações regionais (carteira/cidade/estado)
* Separação por snapshots para simular diferentes momentos do mês
* Geração de relatórios acionáveis via IA Generativa

## **PROPOSTA DE VALOR**

### **Relatórios Inteligentes e Acionáveis**

* **Diagnóstico em tempo real** baseado em dados do dia
* **Recomendações específicas** por carteira/cidade/estado
* **Priorização automática** baseada em urgência e potencial
* **Linguagem natural** para fácil compreensão e ação

### **Grupos de Foco**


1. **Quase Lá** (em teste atual)
   * Identificação de quick wins (gap ≤ R$300 + ocupação ≥70%)
   * Análise de oportunidades de pricing
   * Recomendações por tempo hábil (início/meio/fim do mês)
2. **Overperformers** (próxima fase)
   * Detecção de metas subdimensionadas
   * Análise de erros de classificação de categoria
   * Identificação de padrões de sucesso replicáveis
3. **Críticos** (fase final)
   * Diagnóstico de causas do baixo desempenho
   * Análise de bloqueios e operação
   * Priorização de intervenções urgentes

## **DIFERENCIAIS**

* **Abordagem por features**: 8 indicadores validados estatisticamente
* **Inteligência temporal**: Adaptação de recomendações baseado no momento do mês
* **Granularidade regional**: Análise direcionada por carteira/cidade/estado
* **Sem ML complexo**: Baseado em regras claras e IA Generativa

## **ESTÁGIO ATUAL**

**Teste do relatório para grupo Quase Lá**

* Prompt validado com snapshots de diferentes datas
* Relatórios gerados com insights acionáveis
* Estrutura pronta para implementação nos próximos grupos

## **ROADMAP**

### **Curto Prazo**

### **Médio Prazo**

### **Longo Prazo**

* Integração com sistemas de precificação automática(sytem price)
* Expansão para análise preditiva
