# ⚙️ Setup de Ambiente

Agora que você já tem acesso às ferramentas e sabe onde falar com o time, é hora de preparar seu ambiente de desenvolvimento. Essa parte garante que você consiga rodar as aplicações localmente, testar suas mudanças e contribuir com o time sem depender do ambiente de produção.

Os passos são simples e organizados por projeto. Você vai encontrar instruções curtas, focadas no essencial para colocar cada aplicação de pé no seu computador. Sempre que precisar de ajuda (ou se algo não rodar como esperado), não hesite em chamar alguém do time.
