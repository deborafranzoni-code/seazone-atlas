# Dashboard para Apresentações Comerciais

Este documento centraliza a documentação do projeto **BI para Apresentações Comerciais**, desenvolvido para apoiar o time Comercial com dados de mercado confiáveis e análises que embasam apresentações e decisões comerciais.

A documentação está dividida em dois materiais:

* **Documentação do Usuário**, com orientações de uso e interpretação do dashboard.
* **Documentação Técnica**, com a descrição da arquitetura, fontes de dados e tabelas utilizadas.
