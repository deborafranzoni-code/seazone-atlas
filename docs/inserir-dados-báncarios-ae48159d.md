# Inserir dados báncarios

Para ter um registro novo de dados bancários deve-se criar um registro na `Financial_bank_detail`, o novo ID gerado de ser vinculado a um franqueado ou proprietário, dependendo do pedido de suporte.


* **Se for Franqueado:**

*Account Host > Default Bank Details ID*


* **Se for Proprietário**

*account_owner > default_bank_details_id*

* Se precisar vincular a um imóvel:

*property_property >  bank_details_id*
