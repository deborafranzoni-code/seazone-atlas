# Discovery Host Q2

Manter o design do Sápron, realizando pequenos ajustes para impactar rapidamente a experiência do usuário e coletar insights para o desenvolvimento de ferramentas melhores no futuro. Além disso, refatorar alguns comportamentos que prejudicam a usabilidade.

## *==Refatoração da Tela de Controle==*

A tela de Controle atual apresenta problemas de performance, especialmente para franqueados que administram muitos imóveis. Além disso, as ações disponíveis são confusas e, muitas vezes, acabam sendo inutilizadas.

### Tela Default


1. **Visualização Mobile:**\nBotão Filtro\nVisualização semanal\nMudança no Header

   \
2. **Visualização Desktop:**

   Remover scroll lateral da tela\nMudar Header

### Filtro


1. Fazer modal para filtro
2. Filtrar por **imóvel** e **cama quente**
3. Filtrar por bloqueios, concluídos, novas reservas


### Modal Check-in e Check-out

Melhorar Diagramação de informação\nRemover botões/ações\nFazer ações por stepper ou ordem

## *==Refatoração da Tela de Multicalendar==*

A tela Multi Calendar funciona bem, porém, por ser uma ferramenta compartilhada com co-hostings, precisa de uma visualização específica. Além disso, as informações sobre reservas e bloqueios estão confusas.

### Visualização Host x Co Hosting

Retirar informações sensíveis como taxa de limpeza e valor da reserva da visualização de co hosting

### Padronizar buscas como a do controle

Uso de botão filtro

### Melhorar Modal 

Melhorar diagramação e hierarquia de informação, ter botão de desbloqueio
