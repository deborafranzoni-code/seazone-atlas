# Dia 5 - GUIA

**O que será feito?**

* Acompanhamento na implementação de alguma task de **NÍVEL INICIANTE / INTERMEDIÁRIO** que seja necessário realizar integração com api
* Introduzir uma visão geral sobre o **[Insomnia](https://insomnia.rest/download)**
* **[Instalação](https://insomnia.rest/download)** e configuração do Insomnia para fazer requisições HTTP

**Objetivo do treinamento**

* O objetivo desse treinamento é, além de apresentar a ferramenta Insomnia, reforçar novamente na prática todo o processo de implementação de uma nova task : *criação de uma nova branch*, *documentação*, *commits* e *preenchimento do template da PR*

**Exemplo de tasks de NÍVEL INICIANTE / INTERMEDIÁRIO que poderiam ser repassadas:**

* ***Editar username, telefone ou qualquer outra informação que seja relativamente simples de alterar e que necessite fazer requisição a API***
* ***Correções Integração mais simples***
* ***Integração de dados mockados de páginas/modais mais simples***
