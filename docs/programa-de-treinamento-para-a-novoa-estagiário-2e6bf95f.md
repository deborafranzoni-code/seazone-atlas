# Programa de Treinamento para a Novo(a) Estagiário

**Visão Geral:**

Este programa de treinamento é projetado para capacitar você a passar por todas as hard skills da área de forma prática. O treinamento está dividido em 3 ciclos, baseados nos ciclos CRISP-DM adaptados para aprendizado. Cada ciclo terá uma aula introdutória de 1 hora e um desafio prático que simula um contexto real. Você será incentivada a buscar conhecimento adicional utilizando ferramentas de IA interativas, consultando vídeos e artigos, ou trocando experiências com colegas da equipe.

**Notas Finais:**

* Durante os desafios, sinta-se à vontade para utilizar o ChatGPT e outras ferramentas de IA para auxiliar na resolução de problemas.
* Incentivamos a busca por recursos adicionais, como tutoriais, documentações e fóruns.
* A colaboração com colegas é bem-vinda, desde que mantenha a integridade do seu aprendizado.
* Mantenha um registro organizado do seu progresso e dúvidas para discussão nas próximas reuniões.
