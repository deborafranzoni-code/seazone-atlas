# Como indicar um imóvel para locação

[Como indicar um imóvel para locação.pdf](Como%20indicar%20um%20imo%CC%81vel%20para%20locac%CC%A7a%CC%83o%20792dabb7c4a643868c7d570e386700ed/Como_indicar_um_imvel_para_locao.pdf)

Este guia traz o passo a passo de como indicar um imóvel para alugar. Explica como acessar o site, fazer login e navegar até a seção "Indicar". Também traça dois caminhos diferentes para indicar um imóvel para alugar e disponibiliza um formulário para preenchimento com todas as informações necessárias.


1. Acesse o site <https://sapron.com.br/login>
2. Insira seu login e senha e clique em "Entrar".

3. Para indicar imóveis, você pode ir através de dois caminhos: o primeiro é pelo caminho abaixo, onde no canto inferior da página inicial, você deverá clicar em "Indicar".

 ![https://ajeuwbhvhr.cloudimg.io/colony-recorder.s3.amazonaws.com/files/2023-12-13/c2fba6f2-ec77-4dd3-91d9-7b41b00c4b9c/user_cropped_screenshot.jpeg?tl_px=0,0&br_px=1699,278&force_format=png&width=1120.0&wat=1&wat_opacity=0.7&wat_gravity=northwest&wat_url=https://colony-recorder.s3.us-west-1.amazonaws.com/images/watermarks/FB923C_standard.png&wat_pad=523,95](https://ajeuwbhvhr.cloudimg.io/colony-recorder.s3.amazonaws.com/files/2023-12-13/c2fba6f2-ec77-4dd3-91d9-7b41b00c4b9c/user_cropped_screenshot.jpeg?tl_px=0,0&br_px=1699,278&force_format=png&width=1120.0&wat=1&wat_opacity=0.7&wat_gravity=northwest&wat_url=https://colony-recorder.s3.us-west-1.amazonaws.com/images/watermarks/FB923C_standard.png&wat_pad=523,95)


4. Selecione o tipo de indicação que deseja, nesse caso, locação de imóveis será: "Indicar para locação".

 ![https://ajeuwbhvhr.cloudimg.io/colony-recorder.s3.amazonaws.com/files/2023-12-13/3b0d7a1b-f3a4-48c4-ad1b-3e03ea940e7c/ascreenshot.jpeg?tl_px=433,96&br_px=1723,817&force_format=png&width=1120.0&wat=1&wat_opacity=0.7&wat_gravity=northwest&wat_url=https://colony-recorder.s3.us-west-1.amazonaws.com/images/watermarks/FB923C_standard.png&wat_pad=524,277](https://ajeuwbhvhr.cloudimg.io/colony-recorder.s3.amazonaws.com/files/2023-12-13/3b0d7a1b-f3a4-48c4-ad1b-3e03ea940e7c/ascreenshot.jpeg?tl_px=433,96&br_px=1723,817&force_format=png&width=1120.0&wat=1&wat_opacity=0.7&wat_gravity=northwest&wat_url=https://colony-recorder.s3.us-west-1.amazonaws.com/images/watermarks/FB923C_standard.png&wat_pad=524,277)


5. O segundo caminho é na página inicial, selecione a categoria: "Indicar para Locação".

 ![https://ajeuwbhvhr.cloudimg.io/colony-recorder.s3.amazonaws.com/files/2023-12-13/18cf194e-1107-457e-a24a-3e2deeb18592/user_cropped_screenshot.jpeg?tl_px=0,0&br_px=1719,893&force_format=png&width=1120.0&wat=1&wat_opacity=0.7&wat_gravity=northwest&wat_url=https://colony-recorder.s3.us-west-1.amazonaws.com/images/watermarks/FB923C_standard.png&wat_pad=161,234](https://ajeuwbhvhr.cloudimg.io/colony-recorder.s3.amazonaws.com/files/2023-12-13/18cf194e-1107-457e-a24a-3e2deeb18592/user_cropped_screenshot.jpeg?tl_px=0,0&br_px=1719,893&force_format=png&width=1120.0&wat=1&wat_opacity=0.7&wat_gravity=northwest&wat_url=https://colony-recorder.s3.us-west-1.amazonaws.com/images/watermarks/FB923C_standard.png&wat_pad=161,234)


6. Ao ser redirecionado para o formulário de indicação, preencha todos os dados do imóvel a ser indicado.

 ![https://ajeuwbhvhr.cloudimg.io/colony-recorder.s3.amazonaws.com/files/2023-12-13/3d8ca6fc-c02d-4274-bffc-50e5cd8abbfd/ascreenshot.jpeg?tl_px=0,0&br_px=1719,893&force_format=png&width=1120.0&wat=1&wat_opacity=0.7&wat_gravity=northwest&wat_url=https://colony-recorder.s3.us-west-1.amazonaws.com/images/watermarks/FB923C_standard.png&wat_pad=419,298](https://ajeuwbhvhr.cloudimg.io/colony-recorder.s3.amazonaws.com/files/2023-12-13/3d8ca6fc-c02d-4274-bffc-50e5cd8abbfd/ascreenshot.jpeg?tl_px=0,0&br_px=1719,893&force_format=png&width=1120.0&wat=1&wat_opacity=0.7&wat_gravity=northwest&wat_url=https://colony-recorder.s3.us-west-1.amazonaws.com/images/watermarks/FB923C_standard.png&wat_pad=419,298)


7. Leia os termos sobre as indicações e caso concorde, clique em "Li e aceito os termos"
