# Contas e Acessos

## 🚨 Política de Acesso

**==Não serão criadas novas contas individuais no n8n.==**

Cada time possui uma conta compartilhada. Solicite acesso somente se você realmente precisar usar o n8n para criar ou manter automações da sua área.

* Uma nova conta de time deverá ser **solicitada via suporte**, especificando o email do time.
* [Suporte de solicitação](https://webhook-n8n.seazone.com.br/form/suporte-gov)
