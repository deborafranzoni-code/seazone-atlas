# OLX

Documentação dos scrapers de coleta de dados de imóveis da **OLX Brasil**. Atualmente temos um scraper que coleta dados de anúncios de imóveis (venda, aluguel, temporada, terrenos, comércio e lançamentos) de todas as regiões do Brasil.
