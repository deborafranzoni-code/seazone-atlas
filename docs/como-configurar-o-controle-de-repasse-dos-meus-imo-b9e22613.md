# Como configurar o controle de repasse dos meus imo

[Como configurar o controle de repasse dos meus imóveis.pdf](Como%20configurar%20o%20controle%20de%20repasse%20dos%20meus%20imo%2002a09e57c612452dacb487e7624d3b3e/Como_configurar_o_controle_de_repasse_dos_meus_imveis.pdf)

Este guia fornece instruções passo a passo sobre como configurar o controle de distribuição de repasse de saldo de suas propriedades. Seguindo essas etapas, você poderá gerenciar e personalizar facilmente suas preferências de repasse, garantindo uma experiência perfeita com o Seazone.


1. Acesse o site <https://sapron.com.br/login>
2. Insira seu login e senha e clique em "Entrar".
