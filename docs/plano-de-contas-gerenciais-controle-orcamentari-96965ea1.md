* **Criar categorização do orçamento (utilizando o BP como referência para controle).**
  * Visão simplificado do BP, agrupando contas semelhantes.
  * Nem toda saída é presentada no orçamento (repasses para anfitriões, p. ex.)
  * Categoria "Outros" representa as despesa de baixo valor e baixa recorrência, muito provavelmente algumas delas não irão ocorrer novamente.
* **Categorização conceitual das despesas e receitas para cada categorização sugerida.**
* As categorizações, tomando por referência o BP, pode não condizer com a realidade. Por exemplo, todas as despesas de "viagens" foram mapeadas para a SZS, inclusive considerando as da SZI. Assim, todos os gastos com viagens foram agrupadas no orçamento para a SZS, mesmo que no Admsys possam vir para a SZI, apenas para possibilitar a comparação.
* **Validação das categorizações através da comparação dos valores de projetado vs realizado, garantindo que nenhuma saída do Admsys esteja sendo inserida e comparada em nível de grandeza a uma categoria incorreta.**
* **Finalizar Sumário Mensal de Receita \[Origem de dados não está atualizada\]**
* Criar um Sumário Geral que facilite a análise dos dados.
* Análise de desempenho orçamentário:
  * Comparação entre os valores orçados e os reais para identificar desvios.
  * Avaliação das razões para os desvios e implementação de ações corretivas.
  * Análises por setores específicos.
* Análise de Variação:
  * Análise das variações favoráveis e desfavoráveis de receitas e despesas, com detalhamento para investigação das causas.
* Análise de Fluxo de Caixa:
  * Acompanhamento do fluxo de caixa para garantir que haja liquidez suficiente para atender às obrigações financeiras planejadas.
* Descrever os elementos exatamente oq entra em cada categoria.
