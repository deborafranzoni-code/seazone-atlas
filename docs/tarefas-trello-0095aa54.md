# Tarefas || Trello

Todas as tasks (tarefas) referentes ao projeto Sapron são organizadas no Board **"Sapron-Web"** no **[Trello](https://trello.com/)**.

As tarefas são distribuídas pelo **Bernardo Ribeiro** e **Bill (Bruno Benetti)** e atribuídas a um determinado dev.

### Sobre o Board no Trello do projeto

### Listas (colunas)

* **Resources**

  > Onde está informações de acessos (BD, usuários) e Designs do sistema
* **Product Roadmap**
* **Support To-do**

  > São as tasks de suporte (vindas do channel suporte-sapron e/ou sugeridas pelos usuários) e de correções a serem realizadas.
* **To do**

  > São as tasks referentes às novas features a serem feitas durante a Sprint.
* **In Progress**

  > São as tasks que estão em progresso. Ao iniciar uma task você deverá mover o card para esta coluna
* **Testing**

  > Ao finalizar uma task é realizado aberto o Pull Request (PR) para a `main` com a feature/fix. Após abrir o PR, você deverá mover o card referente à tarefa para a coluna Testing para que os outros devs possam testar o seu trabalho feito.
* **Increment (Sprint Review)**

  > Após testado e aprovado o PR é feito o merge do PR na `main` . Quando isto é feito, o card referente àquele PR é movido para essa lista.

### Cards

Seguimos um modelo de card para as tarefas, esse modelo (template) está na coluna "Resources" do board.

Os card devem sempre ter tags para indicar a que se refere. Cada card possui pelo menos 3 tags:


1. **\[task, feature, fix\]**
2. **\[frontend, backend\]**
3. **\[página/feature que se refere a task\]**
