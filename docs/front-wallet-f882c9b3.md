# [FRONT] Wallet

**Pré-requisitos:** 

> * Node 16+
> * npm (p/ gerenciamento de pacotes)


1. Clone o repositório

   ```bash
   git clone https://github.com/seazone-tech/wallet.git
   ```

   \
2. Crie um arquivo `.env` na raiz do projeto

   
:::info
   Esse arquivo contém as variáveis de ambiente necessárias para rodar o projeto. Basta pedir para qualquer dev encaminhar o arquivo atual.

   :::

   \
3. Instale as dependências

   ```bash
   npm install
   ```

   \
4. Inicie o servidor de desenvolvimento

   ```bash
   npm run dev
   ```

   \
5. Acesse o servidor de desenvolvimento

   ```bash
   http://localhost:3000
   ```

   \
