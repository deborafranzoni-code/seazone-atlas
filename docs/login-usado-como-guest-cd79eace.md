# Login usado como Guest

Para voltar a usar o email que foi usado como guest

Encontrar o registro do usuario na Tabela `Account_user`e mudar o email para 

> szn_guest_<email_usado@xyz.com>


Isso acontece quando a pessoa fez alguma reserva no passado, porem no Sapron esse email não é  usado, por isso podemos liberar o email.
