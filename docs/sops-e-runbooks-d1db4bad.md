# 🧰 SOPs e Runbooks

## Descrição:

Documentação de procedimentos operacionais que podem ser aplicados de forma repetível ou recorrente, como restart de serviços, patch de configuração ou diagnóstico técnico. Devem ser claros, rápidos de aplicar e confiáveis. Devem ser atualizados à medida que ferramentas ou processos evoluem.


***

### Template

```none
🛠 [Serviço] - [Nome da Ação ou Procedimento] - SOP

🌍 Ambiente

Produção / Staging / Dev

☁️ Cluster / Conta AWS

Ex: tools /seazone-prod

🎯 Objetivo

Explique de forma objetiva o que esse procedimento faz e quando ele deve ser usado.

🧾 Pré-Requisitos

- Permissões  
- Acesso ao cluster  
- Contexto necessário

📋 Passo a Passo

1. Ação 1  
2. Ação 2  
3. Validação final

✅ Resultado Esperado

Descreva o que deve acontecer após seguir os passos.

🏷️ Tags

#sop #infra #tools #eks
```
